RAVOT — Inloggen met CODE i.p.v. link (scanner-probleem definitief weg)
========================================================================

De magic LINK is vervangen door een 6-cijferige INLOGCODE. Er is geen link meer
in de mail, dus geen enkele mailscanner (Gmail, Outlook SafeLinks...) kan je
inlog nog "opbranden". Dit lost het probleem structureel op.

HOE HET WERKT:
1. Gezin vult e-mailadres in op ravot.be.
2. Krijgt een mail met een code van 6 cijfers (staat ook in het onderwerp).
3. Typt de code in op de site -> ingelogd.
De code is 15 min geldig, werkt één keer, en is beveiligd tegen raden
(max 5 pogingen, dan is de code dood).

BELANGRIJK — LET OP HET SCHUINE STREEPJE:
Dit pakket bevat een nieuwe database-kolom. Je MOET migreren:

  cd /srv/ravot && git pull && docker compose up -d --build
  docker compose exec web flask migrate-db     # voegt magic_tokens.attempts toe (veilig)

BELANGRIJK — RUIM OUDE MAGIC-LINK BESTANDEN OP DE VPS OP:
Deze bestanden zijn verwijderd en mogen ook op de server weg (anders blijven ze
als dode bestanden staan; ze worden niet meer gebruikt maar geven geen fout):
  app/templates/auth/bevestig_login.html
  app/templates/auth/check_mail.html
  app/templates/mail/magic_link.html
Als je met 'git pull' werkt worden verwijderde bestanden automatisch mee
verwijderd, dus normaal hoef je hier niets te doen.
